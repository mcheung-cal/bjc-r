As the title implies, for this section we will be making fractals from Lab 10 in 3D!

For this first exercise, implement the tree fractal block in Beetle Blocks such that the tree grows upward the z axis.

A tree growing upward, is exciting but let's go a step further. The tree at the moment, though now 3D, is very flat. Why not utilize the extra space and add extra sides to the tree to round it out like a real tree? Fill in and implement the 3d tree block so that it can produce a tree with as many sides as you want.
