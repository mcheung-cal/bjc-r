Welcome to the libraries lab! In this lab, we will introduce you to Bokeh, a well documented, versatile Python library for data visualization. It is great for making interactive graphics that can be easily integrated into a web application or dashboard.

We will walk you through how to visualize data with Bokeh, and by the end of this lab, you will have created these two visualizations below for COVID-19 data in the U.S.. The data we are using for this lab come from the COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University. You can check out their repository here. This lab is developed by AIs Yishu Chao and Michelle Cheung, so if you have any questions, comments or concerns about how the datasets are cleaned and created, please do not hesitate to reach out to us.

Just like lab 17, we will be working in Jupyter. So to get started with this lab:

  • Download the starter file and all the datasets you need with this link
  • Go to datahub.berkeley.edu and login using your Calnet credentials.
  • On the top right there is an upload button. Upload all of the files in the "Lab18-Bokeh" folder. There should be 5 datasets and 1 iPython Notebook.
  • Double click on the lab18_bokeh.ipynb file to get started!

As we mentioned above, Bokeh can be easily intergated into an web application since it renders its plots with HTML and JavaScript. Therefore, although we are using an iPython Notebook for the pupose of this lab, you can actually directly output your plots as an HTML file.
